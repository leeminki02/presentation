프랙탈(Fractal)

어떤 도형의 일부와 전체가 비슷한 형태를 띠는 도형
이런 성질을 "자기유사성(self-similarity)"이라고 함.
Benoît B. Mandelbrot(만델브로트,1924~2010)가 처음 사용한 단어

프랙탈 차원

프랙탈 도형의 실수 범위 차원
D = log(number of self-similar pieces) / log(magnification factor)
차원 = log(자기닮음 도형의 개수) / log(확대 비율)
어느 도형의 길이를 x배 크게 하였을 때 그 도형의 면적(부피)이 n배 증가한다면, 그 도형의 차원은 logxn

D = log23

차원의 확장

직선: D = log N / log N = 1
평면: D = log N2 / log N = 2
큐브: D = log N3 / log N = 3
